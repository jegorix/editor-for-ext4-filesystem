# editor-for-ext4-filesystem

Coursework repository for an ext4 filesystem image analyzer and safe metadata editor.

## Contents

- `editor-for-ext4-filesystem/` - C project with `ncurses` interface for inspecting and editing ext4 image metadata
- `documents/` - coursework documentation
- `diagrams/` - diagrams and supporting materials

## Image files

The repository keeps only one sample ext4 image for testing:

- `editor-for-ext4-filesystem/test-ext4.img`

Generated backup images are excluded from version control.
